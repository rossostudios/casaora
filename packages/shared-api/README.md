# @casaora/shared-api

Shared API contract and lightweight request helpers for Casaora apps.

## Includes

- Generated OpenAPI types (`@casaora/shared-api/types`)
- Shared URL + fetch helpers (`@casaora/shared-api/client`)

## Update OpenAPI types

```bash
cd /Users/christopher/Desktop/puerta-abierta/packages/shared-api
npm run generate:types
```

This command expects `/Users/christopher/Desktop/puerta-abierta/api/openapi.yaml`.
